# Amethyst Demo Script - 110 seconds

## 0:00-0:12 - Problem

"Company data is not automatically safe context for an AI agent. It can be stale, contradictory, unapproved, or restricted by policy. Amethyst turns source evidence into governed operational context."

## 0:12-0:32 - Brain overview

Show the seeded Company Brain workspace.

"Sources and documents remain evidence. Claims, metrics, entities, and policies have lifecycle and provenance. Only approved state enters compiled mission context."

## 0:32-0:52 - Context packet

Open one point-in-time packet.

"This packet freezes the approved claims, policies, missing information, stale context, and contradictions used for one task. Later source changes do not rewrite the context behind an earlier decision."

## 0:52-1:15 - Policy and approval

Open an approval packet.

"Policy is enforced server-side, so a direct API call cannot bypass a deny or review requirement. Every decision emits an immutable receipt."

## 1:15-1:35 - Learning

Show candidate, approval, and writeback.

"Agents cannot silently teach Company Brain. A result proposes a learning candidate. Approval freezes it. A separate idempotent action writes it back."

## 1:35-1:50 - Operations

Show a sanitized operations graphic.

"The internal release passed backup and restore rehearsal, deployed smoke tests, a 40-second rollback, and a full one-hour observation window. External customer access remained a separate authorization gate."
